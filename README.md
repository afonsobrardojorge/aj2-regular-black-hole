# Master development checklist for Buinheira--Sal Jorge / AJ

This checklist tracks what is required for AJ-2/AJ-3 to move from an effective model toward a strong candidate physical theory. It is intentionally strict.

## Current status

AJ-2 is a controlled effective regular-black-hole model. AJ-3 remains a development program, not a completed dynamical theory.

**v0.3.4 note:** this release is a packaging cleanup of v0.3.3: the residual `scripts/__pycache__/` entry was removed and checksums were regenerated.

- [ ] Error estimates and Schwarzschild calibration.
- [ ] Comparison of QNMs with Hayward/Bardeen/Reissner--Nordström-like metrics.
- [ ] Quantitative observational bounds on alpha=a/M and ell/M.
- [ ] External technical review.

- [ ] Derive full field equations.

- [ ] Show whether the AJ-2 density profile is generated dynamically.
- [ ] Derive a^3 = C M ell^2 from the solution, not from an imposed ansatz.
- [ ] Analyze hyperbolicity and effective cones.
- [ ] Prove absence of ghosts and gradient instabilities in the exterior.
- [ ] Derive axial perturbation equations.
- [ ] Derive polar perturbation equations.
- [ ] Study inner-horizon/mass-inflation stability.
- [ ] Develop slow-rotation sector.
- [ ] Test a Kerr-like rotating candidate.
- [ ] Study Vaidya-AJ collapse/accretion.
- [ ] Study semiclassical evaporation trajectories.
- [ ] Incorporate observational constraints.

## Core scientific claim allowed

AJ-2 is a mathematically controlled effective regular-black-hole sector with a finite de Sitter-like core.

## Core scientific claim not allowed

AJ/AJ-2 is not yet a complete physical theory or a final theory of quantum gravity.
